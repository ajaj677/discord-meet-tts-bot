Overview

SpeechFlow is a real-time Text-to-Speech controller that injects synthesized voice directly into Discord Voice Channels and Google Meet sessions.

Key Features

  • Live Text-to-Speech streaming
  • Discord Voice Channel support
  • Google Meet audio injection
  • Session-based controls
  • Real-time activity logs

System Requirements

  • Chrome / Edge browser
  • Discord voice permissions
  • Valid Google Meet link
  • Running backend TTS service

Setup & Configuration

Enter a Google Meet link or Discord Voice Channel ID and click Connect to establish an audio session.

Bot Controls

  • Start – Begin speech streaming
  • Stop – Pause output
  • Restart – Reset the session
  • Disconnect – Terminate connections

Sending TTS Messages

Type your message and click Send. The synthesized voice will play in the connected channel.

Activity Logs

  • INFO – System updates
  • SUCCESS – Completed actions
  • WARNING – Non-critical issues
  • ERROR – Failures

Troubleshooting

  • Check Discord voice permissions
  • Ensure backend service is running
  • Reconnect if status shows disconnected
